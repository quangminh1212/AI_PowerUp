# 🏛️ Open-Weight Models

Top open-source and open-weight AI model repositories — from frontier reasoning to specialized coding and multimodal models.

## Overview

This directory contains **44 submodules** of the world's leading open-weight AI models — covering frontier reasoning (DeepSeek R1, Marco-o1), code generation (CodeLlama, StarCoder2), multimodal vision-language models (LLaVA, CogVLM2), edge-optimized models (MiniCPM, SmolLM), and alternative architectures (RWKV).

## Usage

```bash
# Init a model repository
git submodule update --init --depth 1 models/deepseek-r1

# Run with vLLM inference
pip install vllm && vllm serve deepseek-ai/DeepSeek-R1-Distill-Qwen-7B
```
